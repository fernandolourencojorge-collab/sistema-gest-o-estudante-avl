# sistema-gest-o-estudante-avl
Sistema de Gestão de Estudante usando arvore AVL

Este projeto implementa uma Árvore AVL em Python para gerir estudantes de forma eficiente. Cada estudante é armazenado num nó da árvore, utilizando a matrícula como chave e o nome como valor associado.

A Árvore AVL é uma árvore binária de busca auto-balanceada, o que garante que, após cada inserção, a estrutura permaneça equilibrada. O equilíbrio é mantido através do cálculo do fator de balanceamento e da aplicação de rotações simples e duplas sempre que necessário.

Principais funcionalidades do código:

Inserção de estudantes na árvore AVL

Busca eficiente de estudantes pela matrícula

Manutenção automática do equilíbrio da árvore

Complexidade O(log n) nas operações de inserção e busca

Este sistema demonstra, de forma prática, a aplicação de árvores equilibradas no contexto de um sistema de gestão escolar.
