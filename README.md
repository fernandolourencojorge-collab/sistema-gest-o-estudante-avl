# sistema-gest-o-estudante-avl
Sistema de Gestão de Estudante usando arvore AVL

Este projeto implementa um sistema de gestão escolar utilizando uma árvore AVL,
garantindo eficiência nas operações de inserção, busca,  e remoção de estudantes.

## Funcionalidades
- Inserir estudante
- Buscar estudante por matrícula
- Remover estudante

## Tecnologias
- Python 3
- Estrutura de dados: Árvore AVL

## Como executar
1. Instalar Python 3
2. Executar o ficheiro main.py
